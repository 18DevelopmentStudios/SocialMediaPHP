## S3LT Social Template
# A Fully Intergrated Social Media Site 
previously ran under the names mortal, dynac & vixxy.

- Fully Customisable
- Administration Panel
- Discord Authentication
- Public Chat System
- User Profile Item Shop
- Full Profile Page
- User Searchbox
- Events Page
